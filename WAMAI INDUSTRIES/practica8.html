Primeros pasos en JavaScript


Bien, es hora de comenzar a programr en javascript. Lo primero que tenemos que hacer es crear una plantilla ne html para poder colocar alli nuestro codigo en java script.


Creando la plantilla en html para programar:


1.Abre tu editor de texto favorito como el bloc de notas en windows o gedit en linux o cualquier otro.
2.Escribe el siguiente codigo html:


<html>
<head>
<title>Primer programa en JavaScript</title>
</head>
<body>
<script language=”javascript”>
<!– AQUI IRA TODA TU PROGRAMACION EN JAVASCRIPT –>
</script>
</body>
</html>


3.Guarda el archivo que creaste en un fichero con el nombre que desees, solo asegurate que debes colocarla extension html (htm tambien funciona) para que el archivo sea reconocido como un archivo de html y puedas ver tus programas en javascript.


Primer programa de JavaScript


Ahora bien para nuestro primer programa podemos desplegar un simple mensaje de hola mundo, para hacerlo puedes usar el siguiente codigo.


<html>
<head>
<title>Primer programa en JavaScript</title>
</head>
<body>
<script language=”javascript”>
alert('hola mundo');
</script>
</body>
</html>


Resultado:




La funcion alert despliega una ventana de aviso, con el texto que tengamos dentro.
!Felicidades¡ Si has seguido las instrucciones, guarda tu archivo html, y ejecutalo en tu navegador web, ya has creado tu primer programa en javascript.
